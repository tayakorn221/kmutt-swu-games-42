# 🏸 เว็บติดตามผลทีม มจธ. — มศว เกมส์ ครั้งที่ 42

เว็บไซต์ไฟล์เดียว แสดงโปรแกรม/ผลแบดมินตันของทีมบุคลากร **มจธ. (KMUTT)**
เปิดได้ทั้งมือถือและคอม · อัปเดตผลผ่าน Google Sheets ได้หลายคน

---

## 📁 ไฟล์ในโฟลเดอร์นี้

| ไฟล์ | คืออะไร | ต้องแตะไหม |
|------|---------|-----------|
| **index.html** | ตัวเว็บไซต์ | เปิดอันนี้เพื่อดู |
| **data.js** | ข้อมูลโปรแกรมสำรอง (เว็บใช้ตอนยังไม่เชื่อม Sheet) | ไม่ต้อง |
| **kmutt_sheet.csv** | ตารางสำหรับ import เข้า Google Sheets | ใช้ตอนตั้ง Sheet |
| **มจธ_มศวเกมส์42.xlsx** | ไฟล์ Excel 3 ชีต (ตารางแมตช์ · นักกีฬา · สรุป) | ใช้ดู/พิมพ์ |
| **kmutt_data.json** | ข้อมูลดิบโครงสร้างเต็ม | สำหรับนักพัฒนา |
| extract.py / build_excel.py / build_site.py | สคริปต์ดึง & สร้างไฟล์ใหม่ | เฉพาะตอนรีเฟรชข้อมูล |

## 📊 เชื่อม Google Sheets เพื่ออัปเดตผลเอง (หลายคนช่วยกันได้)

### ครั้งแรก — ตั้งค่า Sheet
1. เข้า [sheets.google.com](https://sheets.google.com) → สร้างไฟล์เปล่า
2. เมนู **ไฟล์ → นำเข้า → อัปโหลด** → เลือก **kmutt_sheet.csv**
   → เลือก "แทนที่สเปรดชีต" แล้วนำเข้า
3. กดปุ่ม **แชร์** (มุมขวาบน) → เปลี่ยนเป็น **"ทุกคนที่มีลิงก์"** สิทธิ์ **ผู้มีสิทธิ์ดู** → คัดลอกลิงก์
4. เปิดเว็บ → ปุ่ม **⚙ เชื่อม Google Sheet** → วางลิงก์ → **บันทึก & เชื่อมต่อ**

เสร็จแล้ว! เว็บจะดึงข้อมูลจาก Sheet และรีเฟรชเองทุก 3 นาที

### เวลาแข่ง — กรอกผล
แก้ใน Google Sheet ได้เลย 2 คอลัมน์:
- **ผล** : พิมพ์ `ชนะ` / `แพ้` / `บาย` (เว้นว่าง = ยังไม่แข่ง)
- **สกอร์** : เช่น `21-15, 21-18`

เปลี่ยนใน Sheet แล้วเว็บอัปเดตตามอัตโนมัติ (หรือกดปุ่ม **↻ รีเฟรช**)

> 💡 อย่าสลับ/ลบคอลัมน์ — แก้แค่ช่อง **ผล** กับ **สกอร์** พอ
