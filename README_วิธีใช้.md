## 🖼️ การ์ดสรุปแมตช์ประจำวัน (สำหรับงานประชาสัมพันธ์)

หน้า **summary.html** สร้างรูปการ์ดสรุปแมตช์ทีม มจธ. ขนาด **1080×1350** (พอดี IG/FB แนวตั้ง)
โพสต์ลงโซเชียลได้รูปเดียวจบ — เช้าโพสต์โปรแกรม ค่ำโพสต์ผล

**วิธีใช้:**
1. เปิด **summary.html** (ต้องอยู่โฟลเดอร์เดียวกับ `data.js` + `summary-fonts.js`)
2. เลือก **วันที่** และโหมดจากแถบเครื่องมือด้านบน
   - **เช้า · พรีวิว** = โชว์ เวลา-คู่แข่ง-คอร์ต-ประเภท (ยังไม่มีผล)
   - **ค่ำ · ผล** = เติมสกอร์ + ไฮไลต์ฝั่งชนะ (แมตช์ที่ยังไม่จบขึ้น "รอแข่ง")
3. กด **⬇ ดาวน์โหลด PNG** → ได้ไฟล์ `summary-<วันที่>-<โหมด>.png`

> โหมดเดียวกันทั้ง 2 แบบ ต่างแค่การโชว์ผล · ฟอนต์ไทยฝังในรูปแล้ว (ไม่เพี้ยนเป็นกล่อง)
> · chip สีประจำมหาวิทยาลัยของคู่แข่ง · ฝั่ง มจธ. ไฮไลต์สีส้ม ★

**ลิงก์ตรง (ใส่ในเบราว์เซอร์ได้เลย):** `summary.html?date=2026-05-31&mode=preview`
หรือบน Vercel: `/summary/2026-05-31?mode=results`

**ทำรูปทั้งหมดทีเดียว (อัตโนมัติ — ไม่บังคับ):**
```
pip install playwright
python -m playwright install chromium
python gen_cards.py            # ได้ทุกวัน × ทั้ง 2 โหมด ไว้ในโฟลเดอร์ cards/
```
